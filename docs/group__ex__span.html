onesdk_result_t onesdk_ex_api_enable_span ( void  )

Initializes the "span" extension API. The SDK must be initialized before using this function.

Returns
ONESDK_SUCCESS if successful, an SDK stub error code otherwise.

The span API is intended to be used for interoperation with OpenTracing and OpenCensus.

Since
This function was added experimentally in version 1.4.0.

void onesdk_ex_spantracer_set_ids ( onesdk_tracer_handle_t  tracer,
onesdk_ex_trace_id_t  trace_id,
onesdk_ex_span_id_t  span_id,
onesdk_ex_span_id_t  parent_span_id 
)
inline

Sets the trace ID, span ID and parent span ID.

Parameters
tracerA valid span tracer handle.
trace_idThe trace ID.
span_id[optional] The span ID associated with this tracer.
parent_span_id[optional] The ID of this tracer's parent span.

If you want to use this function, you must call it before starting the tracer because it creates an incoming link.

Beware: When you parse a trace or span ID from a hex string that is too short, pad with zero from the left.

There is only one trace ID parameter for both the tracer's span ID and the parent's. This is because they must always be equal.
