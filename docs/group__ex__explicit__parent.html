Starts a tracer with the given parent.

Parameters
tracer_handleA valid tracer handle.
parent_tracer_handleA valid handle to a started tracer. The parent must be started. If it has started children, the tracer will be attached to the most recently started (i.e. active) child.

If ONESDK_EX_START_WITHOUT_PARENT is specified a new path will be started with this tracer as root, unless an incoming tag is also set.

If ONESDK_INVALID_HANDLE is specified, the tracer will be started non-capturing.

Parameters
start_time_microStart timestamp of the tracer in microseconds since 1970-01-01 ("Unix Time"). Use ONESDK_CURRENT_TIME to use the current time. BEWARE: Timestamps should not be in the future or more than fifteen seconds in the past. Zero or negative timestamps are reserved for future use and must not be used.
Returns
true if the tracer is capturing, false otherwise.

Starts the specified tracer. This will capture all entry fields of the tracer and start the time measurement.

The tracer will be detached (i.e. not attached to the current thread or any other context).

Use onesdk_tracer_start to start a tracer witht the currently active tracer as parent and attached to the context.
